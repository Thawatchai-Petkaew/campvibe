Sign in with Google — OAuth Login

1 User Journey

เส้นทางผู้ใช้ตั้งแต่กดปุ่มจนเข้าระบบสำเร็จ — Google login วางคู่กับ Credentials เดิม (ไม่แทนที่).

2 ประเภทผู้ใช้

  • ผู้ใช้ใหม่ → Google สร้าง User อัตโนมัติ (role=CAMPER) เหมือน register เดิม
  • ผู้ใช้เดิม (Credentials) → Google อีเมลเดียวกันเข้าบัญชีเดิม (link)

States ที่ต้องครอบ

  • กำลัง redirect (loading) · ยกเลิกที่ Google (กลับ login)
  • error (เช่น email ถูกใช้กับ provider อื่น — ขึ้นกับ D-2)
  • สำเร็จ → Navbar/role อัปเดต (เหมือน Credentials flow)

3 System Design

รายละเอียดทางเทคนิคของแนวทางที่แนะนำ (Approach A — JWT-only).

มิติปัจจุบันหลังเพิ่ม Google (แนะนำ)
ProvidersCredentials เท่านั้นCredentials + Google
Session strategyJWT (ไม่มี adapter)JWT (เหมือนเดิม)
Schema / migration—ไม่เปลี่ยน · ไม่มี migration
Account linking—upsert by email (link)
Session shapeid·email·name·image·roleเหมือนเดิมเป๊ะ
Env vars (ต่อ env)—AUTH_GOOGLE_ID · AUTH_GOOGLE_SECRET (server-only)
Cost—ฟรี

Files ที่แตะ (1 story · ~60–100 บรรทัด)

  • lib/auth.ts — เพิ่ม Google provider + upsert ใน jwt callback
  • lib/actions.ts — server action googleSignIn()
  • app/login/page.tsx — ปุ่ม Google
  • lib/auth.config.ts · middleware.ts — ไม่แตะ
  • docs/adr/ADR-008 — บันทึกการตัดสินใจ JWT-only
  • .env + Vercel (staging/prod) — ใส่ secret

Security & PDPA

  • Secret ใน env เท่านั้น ต่อ env · ห้าม NEXT_PUBLIC_* · rotate เมื่อรั่ว
  • Redirect allowlist ที่ Google Console (กัน open-redirect) + reuse guard เดิมใน actions.ts
  • PDPA: Google ส่ง email/name/avatar = PII (field เดิม) → ต้องอัปเดต privacy policy ก่อนเปิด prod
  • ไม่ต้องแก้ CSP (redirect ระดับ HTTP ไม่ใช่ script)

งานที่เจ้าของทำเอง (~15 นาที, ฟรี)

  • สร้าง Google Cloud project → OAuth Consent Screen (app=CampVibe, scopes openid/email/profile)
  • สร้าง OAuth 2.0 Client ID (Web) → ใส่ Authorized Redirect URIs ต่อ env:
    http://localhost:3000/api/auth/callback/google
    https://campvibe-staging.vercel.app/api/auth/callback/google
    https://<prod-domain>/api/auth/callback/google
  • copy Client ID + Secret → ใส่ env ทั้ง 3
