---
description: how to deploy the application
---

1. Ensure your database is compatible with your host (Postgres for Vercel, SQLite with volumes for Railway).
2. Commit and push your code to a GitHub repository.
3. Link the repository to your chosen hosting provider (Vercel, Railway, or Render).
4. Configure environment variables like `DATABASE_URL`.
5. Run the deployment and verify the live URL.
